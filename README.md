# Plomp

A modern Python module.

## Installation

### Using pip
```bash
pip install plomp
```

### Using poetry
```bash
poetry add plomp
```

### Using uv
```bash
uv install plomp
```

## Development Setup

### Clone the repository
```bash
git clone https://github.com/michaelgiba/plomp.git
cd plomp
```

### Install development dependencies
```bash
# With pip
pip install -e .

# With poetry
poetry install

# With uv
uv install -e .
```
